## VIRAL POST PRINCIPLES — APPLY TO EVERY POST

### 1. LEAD WITH A NUMBER OR RESULT
Never open with context. Open with proof.
Good: "13 emails. 4 campaign types. Under 20 minutes."
Bad: "I've been working on something exciting lately."

### 2. TEACH ONE THING PER POST
One insight. One tip. One result. Not three.
Posts that try to say everything say nothing.

### 3. MAKE THE HOOK WORK ALONE
If someone only reads the first line, they should
still get value or feel something. Test every hook
by reading it in isolation.

### 4. END WITH ENGAGEMENT
Every post ends with a question, a CTA, or a
statement that invites a reply.
Good: "What's the one marketing task you'd automate
first if you could?"
Bad: "Like and share if you found this useful."

### 5. SHOW THE WORK
Where possible, reference a real output, a real
metric, or a real system. Specificity is credibility.
